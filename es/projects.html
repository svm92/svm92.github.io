WHISPER

Escáner de vulnerabilidades en páginas web

WHISPER (Web Hazard & Infrastructure Scanning through automated Pentesting and Reporting) es un escáner de vulnerabilidades para aplicaciones web. Fue desarrollado como trabajo de fin de grado en la Universidad de Castilla-La Mancha.

Escrito en Python. Mediante diferentes herramientas de seguridad, puede escanear una red, detectar vulnerabilidades en aplicaciones web y generar un informe detallando dichas vulnerabilidades, todo ello con mínima intervención por parte del usuario.

Código: https://github.com/svm92/tfg

  • Usa Nmap para escanear e identificar redes
  • Usa OWASP ZAP para aplicar payloads (ataques) contra la aplicación web objetivo
  • Genera un informe (usando GitBook/Jinja2) detallando las vulnerabilidades encontradas
  • Puede controlarse remotamente con una Raspberry Pi mediante VNC (Virtual Network Computing)

Juego de estrategia sin nombre

Juego de estrategia por turnos

Un juego de estrategia para 2~30 jugadores. Cualquier número de esos jugadores puede ser controlado por un humano (localmente) o por una IA.

Cada turno, los jugadores reciben recursos que pueden intercambiar por unidades o ventajas. Las unidades pueden moverse y atacar a otras unidades o bases enemigas.

Código: https://github.com/svm92/Unnamed-Strategy-Game

  • Multijugador local (2~30 jugadores)
  • IA sencilla (toma decisiones basándose en los recursos disponibles, implementa algoritmo de pathfinding A*)
  • Generador procedural de mapas sencillo


Sketched Chroma

Juego 2D de plataformas-puzzles

Juego de plataformas con énfasis en la exploración de múltiples áreas interconectadas.

El jugador puede cambiar de color para resolver ciertos puzzles y acceder a diferentes habilidades.

Código: https://github.com/svm92/2D-Platformer

Enlace: https://justacicada.itch.io/sketched-chroma

  • 8 grandes áreas interconectadas, ofreciendo al jugador cierto grado de libertad sobre el orden en el que superarlas
  • 20+ enemigos "jefe" con comportamientos más complejos o IAs sencillas


Random Number God

Juego de cartas

Juego de cartas y estrategia. El jugador se enfrenta a enemigos usando "dados" (generadores de números aleatorios) para lograr ciertos efectos. Una baraja personalizable de cartas le permite al jugador cambiar los efectos de los dados o alterar las probabilidades de ciertas tiradas.

Código: https://github.com/svm92/RNG-Game

Enlace (Windows, Linux): https://justacicada.itch.io/random-number-god

Enlace (Android): https://play.google.com/store/apps/details?id=samuelVazquez.randomNumberGod

  • Multi-plataforma (Windows, Linux, Android)
  • Baraja editable de cartas. Cartas generadas proceduralmente
  • Enemigos generados aleatoriamente combinando diferentes partes, resultando en diversas combinaciones de habilidades y aspectos visuales
  • El jugador puede progresar indefinidamente, con enemigos progresivamente más difíciles

Solucionador de Sudoku

Programa capaz de resolver puzzles Sudoku, escrito en Java.

Usa una combinación de algoritmos de propagación de restricciones y backtracking para rápidamente encontrar la solución de cualquier puzzle válido. También implementa métodos más lentos y menos eficientes como backtracking puro o algoritmos genéticos, para comparar.

Puede conectar con una base de datos MySQL local para guardar datos acerca de los tiempos y métodos de resolución, permitiendo un posterior análisis.

  • Resuelve un Sudoku estándar 3x3 en una mediana de tiempo de 20ms
  • Puede resolver Sudokus de cualquier tamaño (2x2, 4x4, etc.)
  • Conecta con una base de datos MySQL para guardar / analizar información estadística (para comparar tiempos de resolución usando diferentes algoritmos, etc.)


Generador de Música Procedural

Generador procedural sencillo de archivos de audio MIDI, escrito en Python. Tras analizar una amplia muestra de archivos MIDI, es capaz de aplicar análisis estadístico (usando cadenas de Markov) para generar nuevos archivos MIDI.

Más específicamente, recoge información estadística sobre el orden y abundancia de las notas musicales y tempos para cada instrumento, y entonces elige instrumentos al azar y crea nuevas pistas de audio MIDI basándose en la información recogida.

  • Aplica técnicas básicas de machine learning
