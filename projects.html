WHISPER

Web application vulnerability scanner

WHISPER (Web Hazard & Infrastructure Scanning through automated Pentesting and Reporting) is a vulnerability scanner for web applications. It was developed as a final undergraduate project in the University of Castilla-La Mancha.

It is written in Python. Using different security tools, it can scan a network, detect vulnerabilities in web applications, and generate a report detailing those vulnerabilities, all with minimal intervention from the user.

Code: https://github.com/svm92/tfg

  • Uses Nmap to scan and identify networks
  • Uses OWASP ZAP to apply payloads (attacks) on the target web application
  • Generates a report (using GitBook/Jinja2) detailing the found vulnerabilities
  • Can be controlled remotely via Raspberry Pi using VNC (Virtual Network Computing)



VIDEO GAMES


Unnamed Strategy Game

Turn-based strategy game

A strategy game for 2~30 players. Any number of these players can be controlled by a human (locally) or by an AI.

Each turn, players receive resources that they can trade for units or advantages. Units can move and attack other enemy units or bases.

Code: https://github.com/svm92/Unnamed-Strategy-Game

  • Local multiplayer (2~30 players)
  • Simple AI (makes decisions based on available resources, implements A* pathfinding algorithm)
  • Simple map procedural generator

Random Number God

Card game

Strategy/card game. The player fights off enemies using "dice" (random number generators) to achieve certain effects. A customizable deck of cards allows the player to change the effects of the dice or alter the chances of certain rolls.

Code: https://github.com/svm92/RNG-Game

Link (Windows, Linux): https://justacicada.itch.io/random-number-god

Link (Android): https://play.google.com/store/apps/details?id=samuelVazquez.randomNumberGod

  • Multi-platform (Windows, Linux, Android)
  • Editable deck of cards. Cards are generated procedurally
  • Enemies are generated randomly by combining different parts, resulting in different combinations of abilities and visual appearances
  • The player is allowed to progress indefinitely, with enemies always becoming progressively stronger

Sudoku Solver

Program capable of solving Sudoku puzzles, written in Java.

It uses a combination of constraint propagation and backtracking algorithms to quickly find the solution to any valid puzzle. It also implements slower, less efficient methods such as pure backtracking or genetic algorithms, for comparison purposes.

It can connect to a local MySQL database to save data related to completion times and methods, allowing a later analysis.

  • Median solving time of 20ms for standard 3x3 Sudoku
  • Can solve Sudokus of virtually any size (2x2, 4x4, etc.)
  • Connects to MySQL database to collect / analyze statistical information (to compare solving times under different algorithms, etc.)


Procedural Music Generator

Simple procedural generator for MIDI audio files, written in Python. After analyzing a sizeable sample of human-created MIDI files, it can apply statistical analysis (using Markov Chains) to generate new MIDI files.

More specifically, it collects statistical information about the order and prevalence of music notes and tempos for each instrument, and then chooses random instruments and creates new MIDI tracks using the collected information.

  • Applies basic machine learning techniques
